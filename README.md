## KeyBindings:
1. >Esc ->Exit program
2. > W-A-S-D and Q-E    -> Camera translation
3. > Mouse Position X Y -> Camera rotation
4. > Mouse Scroll       -> Increase/Decrease Camera FOV

***Project Build***: x86 or all configuration

***OpenGL***: GLSL3.3, OpenGL32.lib
